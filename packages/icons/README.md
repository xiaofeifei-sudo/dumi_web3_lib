# pelican-web3-lib-icons

Web3 图标集合，包含常见链、钱包、代币的彩色与线性图标，开箱即用。

### 安装

```bash
npm install pelican-web3-lib-icons --save
```

### 使用示例

```tsx
import { BitcoinColorful } from 'pelican-web3-lib-icons';

export default () => {
  return <BitcoinColorful />;
};
```
